Get html with curl into vim buffer

Get entire html page with curl and write it to a buffer in vim. This is useful for when you need to get data from a site and you dont want to reach all the way over to the mouse :)

	 :r !curl http://rest/service --request POST --data @%

Then strip out the tags

REXML could not parse this XML/HTML: 
 <pre>
	 :%s/
